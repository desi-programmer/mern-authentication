# `Info`

MERN Authentication.

Mongo-Express-React-Node-Bootstrap 

# `Usage`

    Layout : Bootstrap 5
    Icon - react-ionicons
    Form Validation - Formik and Yup

# `Using this`

```bash
 git clone https://github.com/desi-programmer/mern-authentication.git
 cd mern-authentication/client
 npm install
 cd ..
 cd server
 npm install redis
 npm install
 # run seperately or with concurrency
```
